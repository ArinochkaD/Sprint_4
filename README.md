# Sprint_4

Добавлены тесты:

1) Проверка добавления двух книг.

2) Проверка добавления жанра к книге.

3) Проверка получения жанров книгам.

4) Проверка отсутствия книг с возрастным рейтингом в списке книг для детей.

5) Проверка получения книг с определенным жанром.

6) Проверка словаря (books_genre).

7) Проверка добавления в избранное.

8) Проверка получения списка избранных книг.

9) Проверка удаления книги из списка избранных книг.

В том числе использована фикстура (multi_collector) во многих тестах для оптимизации строчек кода и параметризация для удобного тестирования нескольких кейсов.
